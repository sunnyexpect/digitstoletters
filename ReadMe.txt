#Guideline

This project base on  Springboot structure, run DigitstolettersApplication then access the below link with inputData:
http://localhost:8080/digits?inputData=1,2,3
For the inputData parameter, you can input with array number like: 1,2,3 or 4,5,6 etc;
Or you can input with 0-99 like: 34 or 99 etc.
